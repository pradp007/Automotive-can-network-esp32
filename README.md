# Automotive CAN Network Using ESP32 and FreeRTOS

## Overview

This project simulates a multi-node automotive ECU network using ESP32 microcontrollers and MCP2515 CAN modules.

The system consists of:

- CAN1 – Engine ECU
- CAN2 – Dashboard ECU
- CAN3 – Diagnostic ECU

Vehicle parameters such as speed, RPM, and battery voltage are transmitted over a CAN bus and displayed in real time.

## Features

- Multi-node CAN communication
- ESP32-based ECU architecture
- FreeRTOS task scheduling
- OLED dashboard display
- RGB matrix diagnostics
- Headlight control simulation
- Real-time monitoring

## Hardware

- ESP32
- MCP2515
- OLED Display
- RGB Matrix
- Push Buttons
- Potentiometer

## Technologies

ESP32, CAN Bus, FreeRTOS, Embedded C, SPI, I2C
